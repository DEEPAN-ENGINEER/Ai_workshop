# Portfolio Website (Static HTML/CSS)

A clean, modern, and responsive personal portfolio website built using only HTML5 and CSS3.

## Features
- **No JavaScript**: Pure HTML/CSS implementation.
- **Responsive Design**: Mobile-first approach using Flexbox and Grid.
- **PWA Support**: Installable with offline fallback.
- **Clean aesthetically**: Blue & Gray color scheme.

## Project Structure
```
portfolio-website/
├── index.html          # Home
├── about.html          # About Me
├── skills.html         # Skills
├── projects.html       # Projects
├── contact.html        # Contact
├── css/
│   └── style.css       # Main Stylesheet
├── assets/             # Images and Icons
├── manifest.json       # PWA Manifest
└── offline.html        # Offline Page
```

## How to Run
1. Open `index.html` in any modern web browser.
2. No server required (it's a static site).

## Customization
- **Colors**: Edit the variables in `css/style.css` (root).
- **Content**: Update the HTML files with your own information.
